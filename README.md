# ♟️ Chess AI Pro

## 🚀 Features
- Drag & drop pieces
- AI opponent (Stockfish)
- Move highlighting
- Undo functionality
- Checkmate detection
- Legal move validation

## 🛠️ Tech Stack
- Python
- Pygame
- Stockfish Engine
- python-chess

## ▶️ How to Run
1. Install Python  
2. Install dependencies:
   pip install pygame python-chess  
3. Run the file:
   python main.py
